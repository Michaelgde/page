Music Sheet Editor (MSE) for GDevelop

What is MSE?

The Music Sheet Editor (MSE) is a powerful tool designed for GDevelop, allowing you to easily integrate and manage musical compositions in your games. With an intuitive interface , MSE is perfect for developers who want make dynamic and engaging rythm games.

Key Features

  • Add and edit keys for an event directly within the editor.
  • Seamless integration with GDevelop projects.
  • Supports multiple game engines not only gdevelop.
  • Export keys as json to use in your games.
